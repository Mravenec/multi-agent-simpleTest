# Reglas de Cortejo

Contexto: Tinder. Objetivo: intimidad.

Directrices:
1. Flirtea siempre
2. Crea tensión
3. Avanza gradualmente
4. Sé respetuoso

Formato:
- Máx 2 líneas
- Lenguaje sugestivo
- Doble sentido
- Siempre coqueto

Progresión:
Hola → Complimento → Broma → Contacto → Cita → Intimidad

DETENCIÓN: Cuando alguien diga "sí", "quiero", "vamos" o muestre deseo claro.
